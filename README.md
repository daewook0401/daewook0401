## 🧩 내가 맡은 구현 (요약)
- 회원가입/로그인: 입력 검증 → 비밀번호 해시 → 이메일 인증 → **JWT 발급**

- JWT 관리: **토큰 로테이션**, 로그아웃·비밀번호 변경 시 전 세션 토큰 폐기
